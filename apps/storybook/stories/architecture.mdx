## Guiding architecture choices

<CardGrid>
  <Card title="Monorepo by design">
    The repository uses pnpm workspaces and Turborepo so shared UI, testing, analytics, auth, and monitoring code can evolve with coordinated type safety.
  </Card>
  <Card title="Explicit package boundaries">
    Applications, shared packages, infrastructure config, and testing utilities are separated so ownership remains clear and imports stay intentional.
  </Card>
  <Card title="Strict TypeScript">
    Strict mode plus additional flags make cross-package refactors safer and keep the public surface of shared packages honest.
  </Card>
</CardGrid>

## Primary decisions and rationale

<DataTable
  headers={['Decision', 'Why it exists', 'Trade-off']}
